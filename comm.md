基础命令
================
使用 <kbd>win</kbd>+<kbd>r</kbd> 快捷键打开运行，输入对应的命令

**常用命令**

|  命令   | 说明  | 其他 |
| ----  | ----  | --- |
| choice | 类似switch case |
| cmd | dos控制台 | |
| ----  | ----  | --- |

| calc.exe | 计算器 |
